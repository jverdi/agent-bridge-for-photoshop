## Prerequisites

- Node.js 20+ and npm
- Adobe Photoshop 24+

## Add the skill (Codex / Claude Code)

```bash
npx skills add https://agent-bridge-for-photoshop.jaredverdi.com
```

This uses the hosted [skill.md](https://agent-bridge-for-photoshop.jaredverdi.com/skill.md) for Agent Bridge for Photoshop.

If you want to develop Agent Bridge for Photoshop itself, follow [developer setup](/contributing/developer-setup).

### Install CLI from npm

```bash
npm install -g @jverdi/agent-bridge-for-photoshop
```

### Start bridge daemon

```bash
psagent bridge daemon
```

### Install Photoshop plugin from Creative Cloud Desktop

1. Open Creative Cloud Desktop.
2. Navigate to `Stock & Marketplace > Plugins`.
3. Search for `Agent Bridge for Photoshop`.
4. Install the plugin.
5. Open Photoshop, then open the `Agent Bridge` panel.
6. Bridge should auto-connect; if not, click `Connect Bridge`.

### Ask your agent to run Agent Bridge for Photoshop tasks

```text
Create a PSD for an ecommerce product page using the Agent Bridge for Photoshop skill
```
